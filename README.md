# 🌦️ Weather Dashboard

A fast, responsive weather dashboard built with ReactJS that provides real-time and historical weather insights using the Open-Meteo API.

---

## 🚀 Overview

This project automatically detects the user's location using browser GPS and displays detailed weather data including current conditions, hourly forecasts, and historical trends.

The focus of this project is **performance, clean UI, and structured data visualization**.

---

## ✨ Features

### 📍 Location Detection

* Auto-detects user location using browser geolocation
* Fetches weather data based on coordinates

---

### 🌤️ Current Weather

* Current, Min, and Max Temperature
* Humidity, Precipitation, UV Index
* Wind Speed & Surface Pressure
* Sunrise & Sunset timings

---

### 🌫️ Air Quality

* PM2.5 and PM10 levels
* Air Quality Index (AQI)
* Clean visual representation

---

### 📊 Hourly Forecast

* Temperature (°C ⇄ °F toggle)
* Humidity
* Precipitation
* Wind Speed
* Visibility
* PM2.5 & PM10 combined graph

---

### 📅 Historical Analysis

* Select custom date range (up to 2 years)
* Temperature trends (Min / Max / Mean)
* Precipitation totals
* Wind trends
* Air quality trends

---

### 📱 Responsive Design

* Fully mobile-friendly
* Adaptive layout for different screen sizes

---

## ⚡ Performance

* Optimized API calls for fast loading
* Efficient state management
* Smooth rendering with minimal re-renders

---

## 🛠️ Tech Stack

* ReactJS (Vite)
* Open-Meteo API
* Axios
* Recharts / Chart.js
* date-fns
